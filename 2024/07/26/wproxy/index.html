wproxy 代理

1.wproxy 电脑设置代理

  • 默认端口：8899
  • 启动并指定端口：w2 start (-p 8899)
  • 停止：w2 stop
  • 代理设置：系统设置=>网络=>wifi=>详细信息=>代理=>https 或 chrome://settings/?search=%E4%BB%A3%E7%90%86
  • 设置 https：服务器 127.0.0.1 端口 8899
  • 过滤请求：底部 filter
  • 设置代理：切换 Rules 可请求替换，https://www.xiaoyiduoduo.com/ https://www.baidu.com
  • 安装 Charles：端口会默认修改为 8888
  • 官方文档
  • 安装根证书

2.wproxy 手机设置代理

  • 无线局域网=>配置代理=>服务器 172.20.104.45(电脑 ip) 端口 8899
  • 使用 safari 浏览器=>http://172.20.104.45:8899=>HTTPS=>Download RootCA
  • 通用=>VPN 与设备管理=>安装证书
  • 通用=>关于手机=>证书信任设置=>设置信任
  • 手机需要关闭 vpn，否则无法捕获请求

3.企业微信

  • 安装 npm 包：@wecom/jssdk
  • 使用.register 初始化需要使用的 api
  • 使用.sendChatMessage 可发送微信小程序
  • 企业微信跳转到微信小程序链接需要添加.html

4.兼容鸿蒙

  • 当 url 上同一个参数被多次赋值时，不会默认取后一个值
  • 手势滑动差异，横竖屏响应式适配

5.高德地图 api 耗时过长

  • 根据城市名设置地图中心点：可根据城市名获取对应坐标点，在初始化地图时通过 center 设置中心点坐标
  • 根据城市名获取城市边界坐标点绘制城市边界：添加 loading 动画，确保边界绘制完成再显示地图
