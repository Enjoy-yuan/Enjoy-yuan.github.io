interview 面试题

1.闭包

  • MDN：闭包可以让你从内部函数访问外部函数作用域，函数在每次创建时生成闭包

2.原型链

  • MDN：每个实例对象都有一个私有属性（称之为 proto ）指向它的构造函数的原型对象（prototype ）
  • 实践：在 vue 中，在 Vue 的原型上定义的变量和函数，在实例中可以通过 this 来访问

3.跨域

  • MDN：如果两个页面的协议，主机，端口都相同时，称为同源，否则会产生跨域
  • 实践：vue 前端，node 后端，nginx 代理

4.对象的深拷贝和浅拷贝

5.事件循环 Event Loop

  • 宏任务：setTimeout setInterval I/O script
  • 微任务：promise 的回调函数，process.nextTick
  • 微任务队列先执行

6.this

  • MDN：指向当前执行代码的环境对象
  • 在浏览器环境中，默认指向 window
  • 在箭头函数中，指向定义时所在的对象，而不是执行时的对象
  • 使用 new 实例化对象，在构造函数中的 this 指向实例化对象
  • 可使用 call，apply，bind 来绑定 this

7.类数组对象

  • MDN：类似数组的对象，拥有 length 属性
  • 产生条件：arguments，document.getElementsByTagName()，new Set()

8.原型继承

  • Dog 继承 Animal 的所有属性和方法
  • 原型继承缺点：所有实例共用同一个原型，会相互影响

9.new 操作符的作用

10.盒模型

  • 标准盒模型：width = content；box-sizing: content-box;
  • IE 盒模型：width = content + padding + border；box-sizing: border-box;

11.水平垂直居中

  • 直接在父元素上设置 flex 布局，并设置子元素水平垂直居中
  • 在父元素上设置 flex 布局，在子元素上设置 margin: auto;
  • 使用子绝父相，通过 translate 移动子元素

12.块级格式上下文 BFC

  • 定义：一个隔离的独立容器，容器内的子元素不会影响外面的元素
  • 常用创建 BFC：设置 position: absolute;或者 display: flex;或者 overflow: hidden;
  • BFC 的作用：避免相邻的 Box margin 重叠

13.防抖

14.节流

15.数据类型

  • 基本类型：undefined，null，string，number，boolean，symbol，bigInt
  • 引用类型：object
  • 基本类型存放在栈中，引用类型同时存放在栈和堆中，栈中保存变量标识符和指针
  • 引用类型存放地址不同，不能直接比较是否相等，可以使用 String()先将它们转成字符串再来比较
